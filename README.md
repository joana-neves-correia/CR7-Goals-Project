# CR7-Goals-Project
Dataset and analysis of Cristiano Ronaldo's career goals, including match details, clubs, competitions, and visualizations to explore scoring patterns and trends.

## License
This project is for educational purposes. Feel free to use and adapt the dataset for learning and analysis.

## Description
This project contains a dataset and analysis of Cristiano Ronaldo's career goals, including match details, clubs, competitions, and visualizations. The goal is to explore scoring patterns, trends, and insights throughout his career.

## Dataset
The dataset includes the following information:
- Date of the match
- Opponent team
- Competition
- Club / National team
- Goal type and minute
- Match result
- Other relevant details

## Objectives
- Analyze Ronaldo's goal-scoring patterns over time.
- Compare performance across clubs and competitions.
- Visualize trends and milestones in his career.
- Extract actionable insights from the data.

## Tools & Technologies
- Python (Pandas)
- Power BI 
- Excel (data cleaning and preparation)
